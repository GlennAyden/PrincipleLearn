# ========================================
# SETUP INSTRUCTIONS
# ========================================

# 1. Install Node.js (versi 18+)
# 2. Clone repository
# 3. cd principle-learn
# 4. npm install
# 5. Buat file .env.local dengan environment variables di atas
# 6. Setup database PostgreSQL
# 7. npx prisma generate
# 8. npx prisma db push
# 9. npm run dev

# ========================================
# TROUBLESHOOTING
# ========================================

# Jika ada error "Module not found":
# rm -rf node_modules package-lock.json
# npm install

# Jika ada error database:
# - Cek PostgreSQL berjalan
# - Cek DATABASE_URL benar
# - Cek database principle_learn sudah dibuat

# Jika ada error OpenAI:
# - Cek OPENAI_API_KEY benar
# - Cek ada credit di akun OpenAI
# - Cek internet connection
